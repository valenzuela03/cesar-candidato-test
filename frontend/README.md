# Frontend

## Instalación y Ejecución

Desde la carpeta del proyecto front-end:

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Levantar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

Asegurarse de que el Backend esté corriendo (puerto 3000 por defecto) para que la aplicación pueda iniciar sesión y cargar datos.
