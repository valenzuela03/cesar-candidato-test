# Backend

## Instalación y Ejecución

Desde la carpeta del proyecto back-end:

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Levantar el servidor en modo desarrollo:**
   ```bash
   npm run dev
   ```

Esto instalará las dependencias y levantará el servidor de la API en modo desarrollo (usando Nodemon para recarga automática).

## Configuración

Asegúrate de tener configurado tu archivo `.env` o las variables de entorno para la conexión a la base de datos (ver `docker-compose.yml` para referencias de puertos y credenciales por defecto).
